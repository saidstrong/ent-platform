# XY Assistant V2 � UX Copy

## Default helper message (first open)
**EN:**
Hi! I can help with learning questions. You can ask anything, or add sources to ground the answer.
Examples:
- "Summarize the main idea of this lesson."
- "List available courses."
- "Explain this concept in simple terms."

**KZ:**
Сәлем! Мен оқу және платформа туралы сұрақтарға көмектесемін.
Кез келген сұрақ қойыңыз немесе жауапты PDF/транскриптке негіздеу үшін Дереккөз қосу батырмасын қолданыңыз.
Мысалдар:
- “Қол жетімді курстарды көрсет.”
- “10-тараудың негізгі идеясын қысқаша түсіндір.”
- “Осы ұғымды қарапайым тілмен түсіндір.”

No sources selected. I can answer generally, or you can add sources to ground the response.

**KZ:**
Дереккөз таңдалмады. Мен жалпы түрде жауап бере аламын немесе жауапты негіздеу үшін дереккөз қосыңыз.
