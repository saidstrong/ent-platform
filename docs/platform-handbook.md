# XY-School Platform Handbook (v1.1)

## What XY-School is
XY-School is a bilingual (EN/KZ) learning platform focused on high?quality learning experiences for web and mobile. It supports structured courses, lessons, and an AI assistant to guide learning.

## Roles & responsibilities
- **Student**: browse courses, enroll, learn lessons, use the AI assistant for help.
- **Teacher**: create/manage courses & lessons, upload resources (PDF), add YouTube videos, optionally attach transcripts, add lesson AI context (EN/KZ), set assistance rules/policy.
- **Admin**: platform owner/operator, manages support, teacher/admin operations, handles incidents.

## Navigation map (high-level pages)
- **Course catalog**: courses list.
- **Course learn page**: course overview and module/lesson structure.
- **Lesson page**: lesson content + resources.
- **Dashboards**: student / teacher / admin (role-based access).

## Content types
- **Course metadata**: title, description, objectives.
- **Lesson content**: text, embedded media, and optional AI context (EN/KZ).
- **Resources**:
  - PDF files (Storage)
  - YouTube videos (optional transcript text)
  - Lesson AI context (EN/KZ)

## Language behavior
- Users can operate in EN/KZ.
- The assistant should answer in the user’s selected language when possible.
